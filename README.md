# ToolVanta

ToolVanta is a fully static, GitHub Pages compatible online tools website. It uses plain HTML, CSS, and JavaScript only. There is no backend, login system, database, build step, paid hosting requirement, or framework dependency.

## What is included

- 150 browser-based tools
- SEO landing pages for every tool
- Category pages and resource workflow pages
- Tool comparison pages for high-intent internal links
- Static `search-index.json` for site search and future enhancements
- Custom `404.html` and `offline.html` pages
- Dark mode, favorites, recently used tools, and PWA support
- `sitemap.xml`, `robots.txt`, `manifest.json`, and `.nojekyll`
- AdSense-friendly placeholder areas
- Privacy, Terms, Cookie Policy, About, and Contact pages

## GitHub Pages deployment

1. Upload all files in this folder to a GitHub repository.
2. Keep `.nojekyll` in the root.
3. In GitHub, open Settings > Pages.
4. Select the branch and root folder.
5. Save and wait for the Pages URL to become active.

## Recommended before publishing

- Replace `toolvanta.space` canonical URLs if you use a different domain.
- Update `CNAME` if your custom domain is different.
- Add your real contact email to legal/contact pages.
- Add AdSense code only after the site is approved.
- Submit `sitemap.xml` in Google Search Console.
